# Filippa

Filippa è un agente AI leggero e configurabile sviluppato in Python.

Il progetto nasce con l'obiettivo di creare un assistente semplice da modificare,
con una struttura pulita e una separazione tra il motore dell'agente, le
configurazioni e i dati.

La documentazione ufficiale è disponibile nella cartella `docs`.

---

# Funzionalità

Filippa include:

- 😀 Sistema emoji configurabile.
- 🐛 Modalità debug per lo sviluppo.
- 🏷️ Nomi personalizzabili.
- 💬 Sistema di risposte tramite dizionari.
- 🧠 Modalità DeepAgent.

---

# Installazione

## Requisiti

- Python 3.x

## Avvio

Esegui:

```bash
python filippa.py
```

Dopo l'avvio, Filippa sarà pronta a ricevere messaggi.

---

# Configurazione

Le principali impostazioni si trovano in `filippa.py`.

Le configurazioni disponibili includono:

- Uso delle emoji.
- Modalità debug.
- Gestione dei nomi.
- Modalità DeepAgent.

Il nome dell'agente e le risposte vengono gestiti tramite `meta.py`.
